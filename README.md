# Naive Bayes Classification Project

## 📌 Deskripsi
Project ini berisi implementasi algoritma Naive Bayes menggunakan Python di Google Colab.

## 📂 Dataset yang digunakan:
1. Iris Dataset (klasifikasi jenis bunga)
2. Dataset Buah Pisang (klasifikasi kematangan)

## ⚙️ Algoritma
Menggunakan Gaussian Naive Bayes dari sklearn

## 🚀 Tools
- Python
- Google Colab
- Scikit-learn

## 📊 Hasil
Model berhasil melakukan klasifikasi dengan akurasi tinggi pada kedua dataset.
